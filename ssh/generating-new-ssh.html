These command are were tested on Ubuntu based linux distro, but should work the same on windows computer.
To run these commands on a windows computer, you might need to install Gitbash or WSL Windows Subsystem for Linux.

Adding your public key to GitHub

Open GitHub settings

Select SSH and GPG keys

click SSH and GPG

New SSH key

click the Green button to start the process

Naming the new token

Give your new SSH token a name to differentiate it from the other keys that you might add in future
After adding all the required data click save, GitHub will prompt you enter your password to confirm your identity
